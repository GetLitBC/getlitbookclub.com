# GetLit Bookclub

A lightweight single-page app for managing a book club with no build step and no dependencies.

## Features

- Track books and mark one as the current club pick
- Add members and update reading progress
- Schedule meetings and toggle RSVP status
- Save discussion prompts and hosting notes
- Persist everything in `localStorage`

## Run it locally

Start the built-in Ruby web server:

```bash
ruby server.rb
```

Then visit [http://localhost:8000](http://localhost:8000).
